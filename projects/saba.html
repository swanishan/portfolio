Sobreuso SABA

An AstraZeneca Mexico website for doctors to educate them about SABA and check if their patients are overusing the medication or not.

Live Website Jump to Prototype

The Project

The aim of this project was to design a website that educates about the overuse of SABA, a class of drugs that is used to treat Asthma. The website was ought to be divided into three sections: general information about Asthma, courses & clinical case studies, and a test section where doctors can check if there patients are overusing or not.

This project focused heavily on the information architecture of the website. The amount of information provided in the website was huge, and the goal was to break it down, organize, and present it in a fashion that is easily digestible.

Project Success

The website was developed by respective professionals at TCS and AZ Mexico. It is live now and can be viewed here.
